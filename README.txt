# Presente Virtual Especial

Site de aniversário virtual feito em HTML, CSS e JavaScript.

## Como usar
1. Abra o arquivo index.html no navegador.
2. Para trocar imagens, substitua os blocos com a classe image-placeholder por tags img.

Exemplo:

<div class="image-placeholder tall">
  <span>Coloque aqui uma imagem</span>
</div>

Troque por:

<img class="foto tall" src="imagens/nome-da-imagem.jpg" alt="Descrição da imagem">
